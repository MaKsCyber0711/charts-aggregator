# Charts Aggregator

Holt aktuelle Charts von FM4, Beatport und anderen Quellen und verlinkt jeden Song direkt zu Apple Music.

## Setup

```bash
pip3 install -r requirements.txt
```

## Ausführen

```bash
python3 charts.py
```

Danach `charts.html` im Browser öffnen.

## Quellen
- 📻 FM4 (orf.at)
- 🎛️ Beatport Top 100
